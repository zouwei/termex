# v0.8.0 -- Plugin System + Extensibility

> Status: **Planned**
> Depends on: v0.6.0

## Goals

开放扩展能力，允许社区通过插件机制扩展 Termex 功能。

## Features

### Plugin Architecture
- [ ] 插件清单格式定义（plugin.json: name, version, permissions, entry）
- [ ] 插件沙箱运行环境（WASM 或受限 JS 执行上下文）
- [ ] 插件生命周期管理（install, enable, disable, uninstall）
- [ ] 插件权限系统（terminal_read, terminal_write, server_info, network 等）
- [ ] 插件 API 暴露（终端数据流、服务器信息、UI 扩展点）

### Plugin Manager
- [ ] 插件列表 UI（已安装 / 可用 / 搜索）
- [ ] 一键安装/更新/卸载
- [ ] 插件设置面板（每个插件独立配置）

### Built-in Plugins (示范)
- [ ] Snippet Manager -- 命令片段管理（保存/搜索/插入常用命令）
- [ ] SSH Config Import -- 从 ~/.ssh/config 导入服务器

### Team Features (Preview)
- [ ] 团队配置共享协议设计
- [ ] 加密配置分发格式

### Tests
- [ ] 插件清单解析测试
- [ ] 插件权限检查测试
- [ ] 插件生命周期测试
