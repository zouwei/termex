# v0.7.0 -- Session Recording + Server Monitoring

> Status: **Planned**
> Depends on: v0.4.0

## Goals

运维增强：终端会话录制与回放，服务器基础监控面板。

## Features

### Session Recording
- [ ] 终端会话录制（asciicast v2 格式兼容）
- [ ] 录制文件存储（本地目录，按日期/服务器组织）
- [ ] 会话回放播放器（速度控制、搜索、跳转）
- [ ] 录制列表管理（查看/删除/导出）
- [ ] 自动录制选项（按服务器/全局配置）
- [ ] 录制文件大小限制 + 自动清理策略

### Server Monitoring
- [ ] 连接后自动采集服务器基础信息（OS, CPU, Memory, Disk）
- [ ] 状态栏实时显示 CPU / 内存使用率
- [ ] 服务器画像卡片（连接详情弹窗）
- [ ] 服务器标签自动识别（Web Server / DB / K8s Node 等）

### AI Session Summary
- [ ] 长时间操作后 AI 自动总结本次会话内容
- [ ] 会话摘要保存到服务器备注

### Tests
- [ ] asciicast 格式写入/读取测试
- [ ] 回放时间轴计算测试
- [ ] 服务器信息解析测试
