# v0.1.0 -- MVP: SSH + Terminal + Server Management

> Status: **Completed**

## Goals

建立核心产品骨架，实现 SSH 连接 + 终端 + 服务器管理的最小可用闭环。

## Features

### Infrastructure
- [x] Tauri v2 + Vue 3 + TypeScript 项目初始化
- [x] Tailwind CSS + Element Plus + Pinia 集成
- [x] vue-i18n 国际化（中文 + 英文）
- [x] 跨平台 CI/CD（GitHub Actions，5 平台构建）
- [x] 版本同步脚本（bump-version.mjs）

### Storage (SQLCipher)
- [x] SQLCipher 加密数据库初始化
- [x] 版本化迁移系统（_migrations 表）
- [x] 7 张数据表：groups, servers, ssh_keys, port_forwards, ai_providers, settings, known_hosts
- [x] 4 个索引

### Crypto
- [x] AES-256-GCM 加密/解密（ring crate）
- [x] Argon2id 密钥派生（m=64MB, t=3, p=4）
- [x] 主密码设置/验证/变更/锁定
- [x] 密码变更时自动 re-encrypt 所有加密字段
- [x] 13 个单元测试

### Server Management
- [x] 服务器 CRUD（create/read/update/delete）
- [x] 分组 CRUD + 嵌套分组
- [x] 服务器/分组排序
- [x] 密码/密钥口令 AES-256-GCM 加密存储
- [x] Pinia serverStore（搜索、分组树、无分组列表）

### SSH Core
- [x] SSH 连接（russh，纯 Rust 实现）
- [x] 密码认证 + 私钥认证（RSA/Ed25519）
- [x] Shell Channel 双向数据流（事件驱动）
- [x] 终端窗口大小调整（window_change）
- [x] 会话管理（connect/disconnect/write/resize）
- [x] 连接状态事件推送（connecting/connected/disconnected/error）

### Terminal (xterm.js)
- [x] xterm.js WebGL 渲染（自动降级 Canvas）
- [x] SSH 数据双向绑定（onData → ssh_write，ssh://data → terminal.write）
- [x] ResizeObserver 自适应容器大小
- [x] 终端主题配色（GitHub Dark 风格）

### UI
- [x] 主布局：侧边栏 + 标签栏 + 终端区 + 状态栏
- [x] 侧边栏：下拉菜单、搜索、分组树、服务器节点、tooltip
- [x] 多标签页：状态指示灯、hover 关闭、切换
- [x] 状态栏：连接状态显示
- [x] 连接对话框：新建/编辑服务器表单
- [x] 设置面板：外观（主题/语言）+ 终端（字体/光标/回滚行数）
- [x] 欢迎页

### Keyboard Shortcuts
- [x] Ctrl+B 切换侧边栏
- [x] Ctrl+N 新建连接
- [x] Ctrl+, 打开设置
- [x] Ctrl+W 关闭当前标签
- [x] Ctrl+Tab / Ctrl+Shift+Tab 切换标签
- [x] Ctrl+1~9 跳转标签
