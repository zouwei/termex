### 合规需求

终端会话录制是金融、医疗、政府等行业的审计合规要求：

- **ISO 27001:2022 A.8.15** — 日志记录（Logging），要求记录用户活动和安全事件
- **等保 2.0 三级 (GB/T 22239)** — 安全审计，要求对关键操作进行审计记录
- **PCI DSS 10.2** — 记录所有系统组件的可追溯审计轨迹
- **SOX Section 404** — IT 操作可审计性

### 目标

1. 完善后端录制引擎 — 按 server/date 归档、自动录制、大小限制
2. 数据库持久化 — 录制元数据入库，支持查询/过滤
3. 前端完整 UI — 录制指示器 + 播放器 + 管理列表
4. AI 会话摘要 — 录制结束后 AI 分析终端内容，生成结构化摘要
5. 自动清理 — 按保留策略回收过期录制

### 2.2 录制大小限制

防止无限增长的录制文件占满磁盘。每次 `record_output` 时检查累积大小：

```rust
struct ActiveRecording {
    header: AsciicastHeader,
    events: Vec<AsciicastEvent>,
    start_time: std::time::Instant,
    file_path: PathBuf,
    recording_id: String,
    server_id: String,
    server_name: String,
    auto_recorded: bool,
    accumulated_bytes: usize,  // 新增：已累积字节数
    max_bytes: usize,          // 新增：最大字节数（默认 50MB）
}

pub async fn record_output(&self, session_id: &str, data: &str) -> bool {
    let mut recordings = self.recordings.write().await;
    if let Some(rec) = recordings.get_mut(session_id) {
        rec.accumulated_bytes += data.len();
        if rec.accumulated_bytes > rec.max_bytes {
            // 返回 false 表示已达上限，调用方应自动停止录制
            return false;
        }
        let elapsed = rec.start_time.elapsed().as_secs_f64();
        rec.events.push(AsciicastEvent::output(elapsed, data));
        return true;
    }
    false
}
```

当 `record_output` 返回 `false` 时，SSH 数据接收处自动调用 `stop()` 并发送前端事件通知：

```
emit("recording://auto-stopped/{session_id}", { reason: "size_limit" })
```

### 2.4 SSH 数据路径集成

录制需要拦截 SSH 输出数据流。当前 SSH data 通过事件发送到前端 — 录制在同一处 hook：

**修改文件**: `src-tauri/src/ssh/session.rs` 中的数据回调

```rust
// 在 SSH channel data callback 中
async fn handle_data(session_id: &str, data: &[u8], state: &AppState, app: &AppHandle) {
    let text = String::from_utf8_lossy(data);

    // 1. 发送到前端终端
    let _ = app.emit(&format!("ssh://data/{session_id}"), data);

    // 2. 录制（如果活跃）
    let within_limit = state.recorder.record_output(session_id, &text).await;
    if !within_limit {
        // 自动停止录制
        if let Ok(path) = state.recorder.stop(session_id).await {
            // 更新 DB: duration, file_size, ended_at, event_count
            finalize_recording(state, session_id, &path).await;
            let _ = app.emit(
                &format!("recording://auto-stopped/{session_id}"),
                serde_json::json!({ "reason": "size_limit" }),
            );
        }
    }
}
```

```rust
/// 开始录制（手动触发）
#[tauri::command]
pub async fn recording_start(
    state: State<'_, AppState>,
    session_id: String,
    server_id: String,
    server_name: String,
    cols: u32,
    rows: u32,
    title: Option<String>,
) -> Result<String, String> {
    let (rec_id, path) = state.recorder
        .start(&session_id, &server_id, &server_name, cols, rows, title, false)
        .await
        .map_err(|e| e.to_string())?;

    // 写入 DB
    let now = chrono::Utc::now().to_rfc3339();
    let meta = RecordingMeta {
        id: rec_id.clone(),
        session_id,
        server_id,
        server_name,
        file_path: path.to_string_lossy().to_string(),
        file_size: 0,
        duration_ms: 0,
        cols,
        rows,
        event_count: 0,
        summary: None,
        auto_recorded: false,
        started_at: now.clone(),
        ended_at: None,
        created_at: now,
    };
    state.db.with_conn(|conn| {
        storage::recording::insert(conn, &meta)
    }).map_err(|e| e.to_string())?;

    Ok(rec_id)
}

/// 停止录制
#[tauri::command]
pub async fn recording_stop(
    state: State<'_, AppState>,
    session_id: String,
) -> Result<RecordingMeta, String> {
    let path = state.recorder.stop(&session_id).await
        .map_err(|e| e.to_string())?;

    // 读取文件信息，更新 DB
    let meta = finalize_recording(&state, &session_id, &path).await?;
    Ok(meta)
}

/// 列出录制（DB 查询，支持分页和过滤）
#[tauri::command]
pub async fn recording_list(
    state: State<'_, AppState>,
    server_id: Option<String>,
    limit: Option<i64>,
    offset: Option<i64>,
) -> Result<Vec<RecordingMeta>, String> {
    let limit = limit.unwrap_or(50);
    let offset = offset.unwrap_or(0);
    state.db.with_conn(|conn| {
        match server_id {
            Some(sid) => storage::recording::list_by_server(conn, &sid, limit, offset),
            None => storage::recording::list_all(conn, limit, offset),
        }
    }).map_err(|e| e.to_string())
}

/// 读取录制文件内容（用于前端回放）
#[tauri::command]
pub fn recording_read(path: String) -> Result<String, String> {
    std::fs::read_to_string(&path).map_err(|e| e.to_string())
}

/// 删除录制（文件 + DB 记录）
#[tauri::command]
pub async fn recording_delete(
    state: State<'_, AppState>,
    recording_id: String,
) -> Result<(), String> {
    // 1. 从 DB 获取 file_path
    let meta = state.db.with_conn(|conn| {
        storage::recording::get(conn, &recording_id)
    }).map_err(|e| e.to_string())?;

    // 2. 删除文件
    if let Some(ref m) = meta {
        let _ = std::fs::remove_file(&m.file_path);
    }

    // 3. 删除 DB 记录
    state.db.with_conn(|conn| {
        storage::recording::delete(conn, &recording_id)
    }).map_err(|e| e.to_string())
}
```

## 五、AI 会话摘要

### 5.1 设计思路

录制结束后（或用户主动点击"生成摘要"按钮），将录制的纯文本内容发送给 AI，生成结构化摘要。

**安全原则**：遵循 CLAUDE.md 中的 "NEVER send credentials to AI providers" 规则 — 仅发送终端输出文本。如果检测到密码提示行（`password:`, `passphrase:`），自动替换为 `[REDACTED]`。

### 5.2 后端实现

**修改文件**: `src-tauri/src/commands/recording.rs`

```rust
#[tauri::command]
pub async fn recording_summarize(
    state: State<'_, AppState>,
    app: AppHandle,
    recording_id: String,
    request_id: String,
) -> Result<(), String> {
    // 1. 从 DB 获取录制信息
    let meta = state.db.with_conn(|conn| {
        storage::recording::get(conn, &recording_id)
    }).map_err(|e| e.to_string())?
        .ok_or("recording not found")?;

    // 2. 读取 .cast 文件并提取纯文本
    let content = std::fs::read_to_string(&meta.file_path)
        .map_err(|e| e.to_string())?;
    let cast = AsciicastFile::parse(&content).map_err(|e| e.to_string())?;

    // 3. 提取输出文本，脱敏处理
    let mut output_text = String::new();
    for event in &cast.events {
        if event.1 == "o" {
            output_text.push_str(&event.2);
        }
    }
    let sanitized = redact_sensitive(&output_text);

    // 4. 截断到合理长度（避免超出 AI token 限制）
    let max_chars = 50_000; // 约 12k tokens
    let truncated = if sanitized.len() > max_chars {
        format!(
            "{}\n\n[... truncated {} characters ...]\n\n{}",
            &sanitized[..max_chars / 2],
            sanitized.len() - max_chars,
            &sanitized[sanitized.len() - max_chars / 2..],
        )
    } else {
        sanitized
    };

    // 5. 调用 AI
    let system = "You are a DevOps expert. Analyze the terminal session recording and produce a JSON summary.\n\
        Output ONLY a JSON object with these fields:\n\
        - overview: string (1-3 sentence summary)\n\
        - commands: array of {command, description, timestamp}\n\
        - errors: array of {error, resolution, timestamp}\n\
        - securityActions: array of strings\n\
        - keyFindings: array of strings\n\
        Use the user's language (Chinese if the session is in Chinese).";

    let user_msg = format!(
        "Session on server '{}', duration: {}s\n\n---\n\n{}",
        meta.server_name,
        meta.duration_ms / 1000,
        truncated,
    );

    let response = call_ai_provider(/* ... */).await
        .map_err(|e| e.to_string())?;

    // 6. 解析并存储摘要
    let summary_json = extract_json_from_response(&response)?;
    state.db.with_conn(|conn| {

    }).map_err(|e| e.to_string())?;

    }));

    Ok(())

/// 脱敏处理：移除密码提示行
fn redact_sensitive(text: &str) -> String {
    let patterns = ["password:", "passphrase:", "Password:", "Passphrase:"];
    let mut result = text.to_string();
    for pat in &patterns {
        // 将密码提示后同一行的内容替换为 [REDACTED]
        // 使用简单启发式：找到 pattern 后，到下一个换行符之间的内容
        while let Some(pos) = result.find(pat) {
            let end = result[pos..].find('\n').map(|i| pos + i).unwrap_or(result.len());
            let replacement = format!("{} [REDACTED]", pat);
            result.replace_range(pos..end, &replacement);
        }
    }
    result
}
```

```vue
<script setup lang="ts">
import { ref, onMounted, onUnmounted, watch } from 'vue';
import { Terminal } from '@xterm/xterm';
import { FitAddon } from '@xterm/addon-fit';
import type { Recording, PlaybackState } from '@/types/recording';

const props = defineProps<{
  recording: Recording;
  castContent: string;
}>();

const terminalRef = ref<HTMLDivElement>();
const playback = ref<PlaybackState>({
  playing: false,
  speed: 1,
  currentTime: 0,
  totalDuration: 0,
  progress: 0,
});

let terminal: Terminal | null = null;
let fitAddon: FitAddon | null = null;
let events: Array<[number, string, string]> = [];
let playbackTimer: ReturnType<typeof setTimeout> | null = null;
let currentEventIndex = 0;

onMounted(() => {
  // 初始化只读 xterm.js 实例
  terminal = new Terminal({
    cols: props.recording.cols,
    rows: props.recording.rows,
    disableStdin: true,  // 只读
    cursorBlink: false,
    scrollback: 10000,
  });
  fitAddon = new FitAddon();
  terminal.loadAddon(fitAddon);
  terminal.open(terminalRef.value!);

  // 解析 asciicast 内容

});

function parseAsciicast(content: string) {
  const lines = content.split('\n');
  // 第一行是 header
  const header = JSON.parse(lines[0]);
  playback.value.totalDuration = 0;

  events = [];
  for (let i = 1; i < lines.length; i++) {
    if (!lines[i].trim()) continue;
    const event = JSON.parse(lines[i]) as [number, string, string];
    events.push(event);
    if (event[0] > playback.value.totalDuration) {
      playback.value.totalDuration = event[0];
    }
  }
}

function play() {
  playback.value.playing = true;
  scheduleNextEvent();
}

function pause() {
  playback.value.playing = false;
  if (playbackTimer) clearTimeout(playbackTimer);
}

function scheduleNextEvent() {
  if (!playback.value.playing || currentEventIndex >= events.length) {
    playback.value.playing = false;
    return;
  }

  const event = events[currentEventIndex];
  const delay = currentEventIndex === 0
    ? 0
    : (event[0] - events[currentEventIndex - 1][0]) * 1000 / playback.value.speed;

  playbackTimer = setTimeout(() => {
    if (event[1] === 'o' && terminal) {
      terminal.write(event[2]);
    }
    playback.value.currentTime = event[0];
    playback.value.progress = event[0] / playback.value.totalDuration;
    currentEventIndex++;
    scheduleNextEvent();
  }, Math.max(0, delay));
}

function seek(targetTime: number) {
  // 重置终端，从头重放到 targetTime
  terminal?.reset();
  currentEventIndex = 0;
  for (let i = 0; i < events.length; i++) {
    if (events[i][0] > targetTime) break;
    if (events[i][1] === 'o' && terminal) {
      terminal.write(events[i][2]);
    }
    currentEventIndex = i + 1;
  }
  playback.value.currentTime = targetTime;
  playback.value.progress = targetTime / playback.value.totalDuration;
  if (playback.value.playing) scheduleNextEvent();
}

function setSpeed(speed: number) {
  playback.value.speed = speed;
  // 如果正在播放，重新调度
  if (playback.value.playing) {
    if (playbackTimer) clearTimeout(playbackTimer);
    scheduleNextEvent();
  }
}

// 搜索：在所有 output 事件中查找文本，返回匹配的时间戳
function searchInRecording(query: string): number[] {
  const timestamps: number[] = [];
  let buffer = '';
  for (const event of events) {
    if (event[1] === 'o') {
      buffer += event[2];
      if (buffer.includes(query)) {
        timestamps.push(event[0]);
        buffer = ''; // 避免重复匹配
      }
    }
  }
  return timestamps;
}

onUnmounted(() => {
  pause();
  terminal?.dispose();
});
</script>
```

```typescript
import type { Recording, RecordingStatus } from '@/types/recording';

export const useRecordingStore = defineStore('recording', () => {

    const map = new Map<string, Recording[]>();

      serverId,
      limit: 200,
      offset: 0,
    });
  }

  /** 开始录制 */
  async function startRecording(

    serverId: string,
    serverName: string,

    rows: number,

      sessionId, serverId, serverName, cols, rows,
    });
    activeRecordings.value.set(sessionId, {

      recordingId,
      startedAt: new Date().toISOString(),

      fileSize: 0,

    await loadRecordings(); // 刷新列表
    return meta;
  }

  /** 删除录制 */
  async function deleteRecording(recordingId: string) {
    await invoke('recording_delete', { recordingId });
    recordings.value = recordings.value.filter(r => r.id !== recordingId);
```

## 十、风险与缓解

### 10.1 大文件回放性能（中风险）

**风险**: 长时间录制（数小时）产生的 `.cast` 文件可能有数十万个事件，前端 `seek()` 需要从头重放所有事件到目标时间点。

**缓解策略**:
1. **索引快照** — 每 1000 个事件保存一次终端状态快照（序列化 xterm.js buffer），seek 时从最近的快照恢复
2. **Web Worker** — 将 asciicast 解析和文本搜索放入 Web Worker，不阻塞主线程
3. **延迟加载** — 大文件按 chunk 读取，不一次性加载全部内容

### 10.2 录制与 SSH 性能（低风险）

**风险**: `record_output` 在 SSH 数据热路径上增加 write lock 操作。

**缓解策略**:
1. 使用 `try_write()` 非阻塞写入，写入失败时丢弃该事件（录制完整性 < 终端响应性）
2. 批量缓冲：每 100ms 批量写入，而非逐事件写入
3. 基准测试：确保录制开启后 SSH 延迟增加 < 1ms

### 10.3 AI 摘要准确性（中风险）

**风险**: 终端输出含大量 ANSI 转义序列，AI 可能无法正确理解。

**缓解策略**:
1. 发送前 strip ANSI 转义序列 — 使用 `strip-ansi-escapes` crate
2. 限制输入长度（50k 字符 ~ 12k tokens）
3. 摘要标记为 "AI generated"，用户可编辑修正

### 10.4 磁盘空间（低风险）

**风险**: 自动录制大量 session 消耗磁盘。

**缓解策略**:
1. 默认 50MB/次上限
2. 默认 90 天自动清理
3. 设置页显示录制总占用空间，可一键清理

### A.1 `finalize_recording` 函数实现

录制停止时调用，读取文件元数据并更新 DB。在 `commands/recording.rs` 中：

```rust
/// 录制结束后更新 DB 元数据（duration, file_size, event_count, ended_at）
async fn finalize_recording(
    state: &AppState,
    session_id: &str,
    file_path: &std::path::Path,
) -> Result<RecordingMeta, String> {
    let now = time::OffsetDateTime::now_utc()
        .format(&time::format_description::well_known::Rfc3339)
        .unwrap_or_default();

    // 读取文件信息
    let file_size = std::fs::metadata(file_path)
        .map(|m| m.len() as i64)
        .unwrap_or(0);

    // 解析 .cast 文件获取 event_count 和 duration
    let content = std::fs::read_to_string(file_path)
        .map_err(|e| e.to_string())?;
    let cast = AsciicastFile::parse(&content)
        .map_err(|e| e.to_string())?;
    let event_count = cast.events.len() as i64;
    let duration_ms = (cast.duration() * 1000.0) as i64;

    // 查找 recording_id（通过 session_id + 正在录制状态）
    let recording_id = state.db.with_conn(|conn| {
        conn.query_row(
            "SELECT id FROM recordings WHERE session_id = ?1 AND ended_at IS NULL
             ORDER BY started_at DESC LIMIT 1",
            rusqlite::params![session_id],
            |row| row.get::<_, String>(0),
        )
    }).map_err(|e| e.to_string())?;

    // 更新 DB
    state.db.with_conn(|conn| {
        storage::recording::update_on_stop(
            conn, &recording_id, file_size, duration_ms, event_count, &now,
        )
    }).map_err(|e| e.to_string())?;

    // 返回完整的 meta
    state.db.with_conn(|conn| {
        storage::recording::get(conn, &recording_id)
    }).map_err(|e| e.to_string())?
        .ok_or_else(|| "recording not found after finalize".to_string())
}
```

```typescript
function handleKeydown(e: KeyboardEvent) {
  switch (e.key) {
    case " ":
      e.preventDefault();
      playback.value.playing ? pause() : play();
      break;
    case "ArrowLeft":
      seek(playback.value.currentTime - 5);
      break;
    case "ArrowRight":
      seek(playback.value.currentTime + 5);
      break;
    case "+":
    case "=":
      setSpeed(Math.min(playback.value.speed * 2, 8));
      break;
    case "-":
      setSpeed(Math.max(playback.value.speed / 2, 0.5));
      break;
    case "0":
      seek(0);
      break;
  }
}

function formatTime(seconds: number): string {
  const m = Math.floor(seconds / 60);
  const s = Math.floor(seconds % 60);
  return `${m.toString().padStart(2, "0")}:${s.toString().padStart(2, "0")}`;
}

const parsedSummary = computed(() => {
  if (!props.recording.summary) return null;
  try {
    return typeof props.recording.summary === "string"
      ? JSON.parse(props.recording.summary)
      : props.recording.summary;
  } catch {
    return null;
  }
});
```

```vue
  const groups: Record<string, { serverName: string; recordings: Recording[] }> = {};
  for (const rec of recordingStore.recordings) {
    if (filterServerId.value && rec.serverId !== filterServerId.value) continue;
    if (!groups[rec.serverId]) {
      groups[rec.serverId] = { serverName: rec.serverName, recordings: [] };
    }
    groups[rec.serverId].recordings.push(rec);
  }
  // 每组按时间倒序
  for (const group of Object.values(groups)) {
    group.recordings.sort((a, b) => b.startedAt.localeCompare(a.startedAt));
```
