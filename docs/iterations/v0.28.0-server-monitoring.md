# v0.28.0 — Server Monitoring Dashboard: 服务器监控仪表盘

> 状态：**计划中**
> 依赖：v0.27.0
> 关联：对标 FinalShell 监控面板，开源 SSH 客户端中唯一的实时监控功能

---

## 背景

### 竞品分析

FinalShell 的核心差异化功能是内置服务器监控面板 — 连接 SSH 后自动展示 CPU、内存、磁盘、网络的实时状态。这一功能使 FinalShell 从单纯的 SSH 客户端升级为轻量级服务器运维平台，是其最大卖点之一。

| 功能 | FinalShell | Termius | Tabby | iTerm2 | **Termex** |
|------|-----------|---------|-------|--------|-----------|
| 实时 CPU 监控 | 有 | 无 | 无 | 无 | **本迭代实现** |
| 实时内存监控 | 有 | 无 | 无 | 无 | **本迭代实现** |
| 磁盘使用率 | 有 | 无 | 无 | 无 | **本迭代实现** |
| 网络流量 | 有 | 无 | 无 | 无 | **本迭代实现** |
| 进程列表 | 有 | 无 | 无 | 无 | **本迭代实现** |
| 无需安装 Agent | 有 | - | - | - | **有** |
| 开源 | 否 | 否 | 是(无监控) | 是(无监控) | **是** |

### 核心定位

本迭代将 Termex 从 SSH 连接工具升级为**服务器运维平台** — 这是开源 SSH 客户端中**首个**内置实时监控功能的产品。

### 设计原则

1. **零 Agent 部署** — 所有指标通过 SSH exec 通道采集，服务器无需安装任何软件
2. **零额外连接** — 复用已有 SSH 会话，不创建新连接
3. **最小开销** — 批量命令单次 round-trip，3 秒默认间隔，轻量 Canvas 渲染
4. **跨平台采集** — 支持 Linux / macOS / FreeBSD 三大平台
5. **优雅降级** — 单条命令失败不影响其他指标，显示 N/A

## 一、数据采集架构

### 1.1 采集策略

连接服务器后，通过 SSH exec 通道（独立于交互式 shell 通道）周期性执行轻量级系统命令，解析输出为结构化指标。

```
┌─────────────────────────────────────────────────────────┐
│ Termex 客户端                                            │
│                                                         │
│  ┌──────────────┐    ┌──────────────┐                   │
│  │ Shell Channel │    │ Exec Channel │  ← 监控专用      │
│  │ (交互式终端)  │    │ (指标采集)    │                   │
│  └──────┬───────┘    └──────┬───────┘                   │
│         │                   │                           │
│         └──────┬────────────┘                           │
│                │                                        │
│         ┌──────▼───────┐                                │
│         │  SSH Session  │  ← 复用同一会话               │
│         │  (russh)      │                                │
│         └──────┬───────┘                                │
└────────────────┼────────────────────────────────────────┘
                 │ SSH 协议
          ┌──────▼───────┐
          │ Remote Server │
          │ (无 Agent)    │
          └──────────────┘
```

### 1.2 采集流程

1. 用户连接服务器，monitor 默认不自动启动（可在设置中开启 auto-start）
2. 用户点击监控按钮或使用快捷键 → 调用 `monitor_start`
3. 后端首次采集：检测 OS 类型 → 选择对应命令集 → 采集 OS 基础信息（一次性）
4. 周期采集（默认 3s）：批量执行命令 → 解析输出 → 发送 `monitor://metrics/{session_id}` 事件
5. 保留最近 5 分钟历史（环形缓冲区），用于图表渲染
6. 断开连接时自动停止采集，清理资源

### 1.3 命令批量化

将所有采集命令用分号拼接为单条命令，通过一次 SSH exec 执行，减少网络 round-trip：

```bash
# Linux 批量命令（约 200 字节，单次 exec）
echo '---CPU---' && cat /proc/stat | head -1 && \
echo '---MEM---' && cat /proc/meminfo | head -8 && \
echo '---DISK---' && df -B1 --output=target,size,used,avail / /home 2>/dev/null && \
echo '---NET---' && cat /proc/net/dev && \
echo '---LOAD---' && cat /proc/loadavg && \
echo '---UPTIME---' && cat /proc/uptime && \
echo '---PROCS_CPU---' && ps aux --sort=-%cpu | head -11 && \
echo '---PROCS_MEM---' && ps aux --sort=-%mem | head -11
```

使用 `---SECTION---` 分隔符拆分各段输出，确保解析可靠。

### 1.4 OS 检测与命令集

**首次连接检测**（一次性执行）：

```bash
uname -s
```

| `uname -s` 输出 | OS 类型 | 命令集 |
|-----------------|---------|--------|
| `Linux` | Linux | `/proc` 文件系统 + `ps aux` |
| `Darwin` | macOS | `top -l` + `vm_stat` + `sysctl` |
| `FreeBSD` | FreeBSD | `sysctl` + `vmstat` + `ps aux` |
| 其他 | 未知 | 尝试 Linux 命令集，失败则降级 |

## 二、指标采集命令

### 2.1 Linux 命令集

**CPU**:
```bash
cat /proc/stat | head -1
# 输出: cpu  user nice system idle iowait irq softirq steal guest guest_nice
# 示例: cpu  374292 6380 168498 8737510 14498 0 7652 0 0 0
# 计算: 需要两次采样的差值
#   total_delta = sum(all_delta)
#   idle_delta  = idle_delta + iowait_delta
#   cpu_percent = 100 * (total_delta - idle_delta) / total_delta
```

**Memory**:
```bash
cat /proc/meminfo | head -8
# 输出 (单位 kB):
#   MemTotal:        8053636 kB
#   MemFree:         1234567 kB
#   MemAvailable:    5678901 kB
#   Buffers:          234567 kB
#   Cached:          2345678 kB
#   SwapCached:        12345 kB
#   SwapTotal:       2097148 kB
#   SwapFree:        2097148 kB
# 计算: used_percent = 100 * (total - available) / total
```

**Disk**:
```bash
df -B1 --output=target,size,used,avail / /home 2>/dev/null
# 输出 (单位 bytes):
#   Mounted on              Size         Used        Avail
#   /              107374182400  64424509440 42949672960
#   /home          214748364800  75161927680 139586437120
# 如果 /home 不存在，只显示 /
# 2>/dev/null 抑制 "No such file or directory" 错误
```

**Network IO**:
```bash
cat /proc/net/dev
# 输出:
#   Inter-|   Receive                                                |  Transmit
#    face |bytes    packets errs drop fifo frame compressed multicast|bytes    packets ...
#     eth0: 123456789  987654    0    0    0     0          0         0 987654321  876543 ...
#     lo:   12345678  123456    0    0    0     0          0         0  12345678  123456 ...
# 过滤: 排除 lo, 匹配 eth/ens/wlan/bond/enp/eno 开头的接口
# 计算: 两次采样差值 / 间隔秒数 = bytes/sec
```

**Load Average**:
```bash
cat /proc/loadavg
# 输出: 0.52 0.38 0.25 1/234 12345
# 解析: 1min=0.52, 5min=0.38, 15min=0.25
```

**Top Processes (by CPU)**:
```bash
ps aux --sort=-%cpu | head -11
# 第 1 行是表头，后 10 行是 top-10 进程
# 输出:
#   USER       PID %CPU %MEM    VSZ   RSS TTY      STAT START   TIME COMMAND
#   root      1234 25.3  4.2 123456 34567 ?        Sl   10:00   1:23 nginx: worker
#   app       5678 18.1 12.5 234567 56789 ?        Sl   09:00   2:34 node server.js
```

**Top Processes (by Memory)**:
```bash
ps aux --sort=-%mem | head -11
```

**Uptime**:
```bash
cat /proc/uptime
# 输出: 123456.78 234567.89
# 解析: 第一个值 = 系统运行秒数
```

**OS Info** (首次采集，一次性):
```bash
uname -srm && cat /etc/os-release 2>/dev/null | head -3
# 输出:
#   Linux 5.15.0-91-generic x86_64
#   NAME="Ubuntu"
#   VERSION="22.04.3 LTS (Jammy Jellyfish)"
#   ID=ubuntu
```

### 2.2 macOS 命令集

**CPU**:
```bash
top -l 1 -n 0 | grep "CPU usage"
# 输出: CPU usage: 12.5% user, 8.3% sys, 79.2% idle
# 直接解析百分比，无需两次采样
```

**Memory**:
```bash
vm_stat | head -10 && echo '---MEMSIZE---' && sysctl -n hw.memsize
# vm_stat 输出 (单位: pages, 1 page = 4096 bytes):
#   Pages free:         123456
#   Pages active:       234567
#   Pages inactive:     345678
#   Pages speculative:  12345
#   Pages wired down:   456789
# hw.memsize 输出: 17179869184 (total bytes)
# 计算: used = (active + wired) * 4096, available = total - used
```

**Disk**:
```bash
df -b /
# 输出:
#   Filesystem   512-blocks      Used Available Capacity  Mounted on
#   /dev/disk3s1 1953595632 654321000 987654000    40%    /
# 注意: 单位是 512-byte blocks
```

**Network IO**:
```bash
netstat -ib | grep -E 'en0|en1'
# 输出:
#   Name  Mtu   Network       Address            Ipkts Ierrs Ibytes    Opkts Oerrs Obytes
#   en0   1500  <Link#7>      xx:xx:xx:xx:xx:xx 123456     0 987654321 654321     0 123456789
# 解析: Ibytes = rx_total, Obytes = tx_total
```

**Processes**:
```bash
ps aux -r | head -11
# macOS 的 ps -r 按 CPU 降序排列
```

**Load Average**:
```bash
sysctl -n vm.loadavg
# 输出: { 0.52 0.38 0.25 }
```

**Uptime**:
```bash
sysctl -n kern.boottime
# 输出: { sec = 1700000000, usec = 0 } ...
# 计算: uptime = now - sec
```

### 2.3 FreeBSD 命令集

**CPU**:
```bash
sysctl -n kern.cp_time
# 输出: user nice system interrupt idle (以 ticks 为单位)
# 与 Linux /proc/stat 类似，需要两次采样计算 delta
```

**Memory**:
```bash
sysctl -n hw.physmem hw.usermem vm.stats.vm.v_free_count vm.stats.vm.v_page_size
```

**其他**: 类似 Linux，`df`/`ps` 命令在 FreeBSD 上基本兼容。

```rust
pub(crate) struct RawCpuCounters {
    pub user: u64,
    pub nice: u64,
    pub system: u64,
    pub idle: u64,
    pub iowait: u64,
    pub irq: u64,
    pub softirq: u64,
    pub steal: u64,
}

impl RawCpuCounters {
    /// 计算总 ticks
    pub fn total(&self) -> u64 {
        self.user + self.nice + self.system + self.idle
            + self.iowait + self.irq + self.softirq + self.steal
    }

    /// 计算空闲 ticks (idle + iowait)
    pub fn idle_total(&self) -> u64 {
        self.idle + self.iowait
    }
}
```

### 3.3 采集器：`src-tauri/src/monitor/collector.rs`

```rust
use std::collections::HashMap;
use std::sync::Arc;
use std::time::Instant;

use chrono::Utc;
use russh::client;
use tauri::{AppHandle, Emitter};
use tokio::sync::{mpsc, RwLock};
use tokio::time::{self, Duration};

use super::parser;
use super::types::*;
use crate::ssh::auth::ClientHandler;

/// 采集器控制命令
pub enum CollectorCommand {
    /// 修改采集间隔
    SetInterval(u64),
    /// 停止采集
    Stop,
}

/// 采集器状态，每个 session 一个实例
pub struct CollectorState {
    /// 控制通道
    pub cmd_tx: mpsc::UnboundedSender<CollectorCommand>,
    /// 后台任务句柄
    pub task_handle: tokio::task::JoinHandle<()>,
}

/// 指标历史环形缓冲区 (最近 5 分钟)
pub struct MetricsHistory {
    /// 固定容量的环形缓冲区
    buffer: Vec<SystemMetrics>,
    /// 写入位置
    write_pos: usize,
    /// 已存储的数量（未超过容量时 == write_pos）
    count: usize,
    /// 缓冲区容量
    capacity: usize,
}

impl MetricsHistory {
    /// 创建新的历史缓冲区
    /// capacity = 5 分钟 / 采集间隔秒数 = 300 / 3 = 100
    pub fn new(capacity: usize) -> Self {
        Self {
            buffer: Vec::with_capacity(capacity),
            write_pos: 0,
            count: 0,
            capacity,
        }
    }

    /// 追加一条指标记录
    pub fn push(&mut self, metrics: SystemMetrics) {
        if self.buffer.len() < self.capacity {
            self.buffer.push(metrics);
        } else {
            self.buffer[self.write_pos] = metrics;
        }
        self.write_pos = (self.write_pos + 1) % self.capacity;
        self.count = (self.count + 1).min(self.capacity);
    }

    /// 获取最新一条
    pub fn latest(&self) -> Option<&SystemMetrics> {
        if self.count == 0 {
            return None;
        }
        let idx = if self.write_pos == 0 {
            self.buffer.len() - 1
        } else {
            self.write_pos - 1
        };
        self.buffer.get(idx)
    }

    /// 获取指定时间范围内的历史记录（按时间升序）
    pub fn range(&self, duration_secs: u64) -> Vec<SystemMetrics> {
        let cutoff = Utc::now().timestamp_millis() - (duration_secs as i64 * 1000);
        let mut result = Vec::new();

        // 按时间顺序遍历环形缓冲区
        let start = if self.count < self.capacity {
            0
        } else {
            self.write_pos
        };
        for i in 0..self.count {
            let idx = (start + i) % self.capacity;
            if self.buffer[idx].timestamp >= cutoff {
                result.push(self.buffer[idx].clone());
            }
        }
        result
    }
}

/// 启动指标采集后台任务
///
/// 复用已有 SSH 会话的 client::Handle 开启 exec 通道进行采集。
/// 每次采集通过单独的 exec channel 执行批量命令，避免干扰交互式 shell。
pub fn spawn_collector(
    handle: client::Handle<ClientHandler>,
    app: AppHandle,
    session_id: String,
    interval_ms: u64,
    history: Arc<RwLock<MetricsHistory>>,
) -> CollectorState {
    let (cmd_tx, mut cmd_rx) = mpsc::unbounded_channel::<CollectorCommand>();
    let sid = session_id.clone();

    let task_handle = tokio::spawn(async move {
        // 1. 检测 OS 类型
        let os_type = detect_os(&handle).await.unwrap_or(ServerOS::Unknown);
        let os_info = collect_os_info(&handle, os_type).await;

        // 发送 OS 信息事件
        let event = format!("monitor://os-info/{sid}");
        let _ = app.emit(&event, &os_info);

        // 2. 初始化前次计数器（CPU delta、Network delta）
        let mut prev_cpu: Option<RawCpuCounters> = None;
        let mut prev_net: HashMap<String, RawNetworkCounters> = HashMap::new();
        let mut prev_time = Instant::now();
        let mut interval = time::interval(Duration::from_millis(interval_ms));

        loop {
            tokio::select! {
                _ = interval.tick() => {
                    let now = Instant::now();
                    let elapsed_secs = now.duration_since(prev_time).as_secs_f64();
                    prev_time = now;

                    // 执行批量命令
                    let batch_cmd = build_batch_command(os_type);
                    match exec_command(&handle, &batch_cmd).await {
                        Ok(output) => {
                            // 解析各段指标
                            let sections = split_sections(&output);

                            let (cpu, raw_cpu) = parser::parse_cpu(
                                os_type,
                                sections.get("CPU").unwrap_or(&String::new()),
                                prev_cpu.as_ref(),
                            );
                            let memory = parser::parse_memory(
                                os_type,
                                sections.get("MEM").unwrap_or(&String::new()),
                            );
                            let disk = parser::parse_disk(
                                os_type,
                                sections.get("DISK").unwrap_or(&String::new()),
                            );
                            let (network, raw_net) = parser::parse_network(
                                os_type,
                                sections.get("NET").unwrap_or(&String::new()),
                                &prev_net,
                                elapsed_secs,
                            );
                            let load = parser::parse_load(
                                os_type,
                                sections.get("LOAD").unwrap_or(&String::new()),
                            );
                            let uptime = parser::parse_uptime(
                                os_type,
                                sections.get("UPTIME").unwrap_or(&String::new()),
                            );
                            let top_cpu = parser::parse_processes(
                                sections.get("PROCS_CPU").unwrap_or(&String::new()),
                            );
                            let top_mem = parser::parse_processes(
                                sections.get("PROCS_MEM").unwrap_or(&String::new()),
                            );

                            // 更新前次计数器
                            prev_cpu = raw_cpu;
                            prev_net = raw_net;

                            let metrics = SystemMetrics {
                                timestamp: Utc::now().timestamp_millis(),
                                cpu,
                                memory,
                                disk,
                                network,
                                load,
                                uptime_seconds: uptime,
                                top_cpu_processes: top_cpu,
                                top_mem_processes: top_mem,
                            };

                            // 存入历史缓冲区
                            {
                                let mut hist = history.write().await;
                                hist.push(metrics.clone());
                            }

                            // 发送到前端
                            let event = format!("monitor://metrics/{sid}");
                            let _ = app.emit(&event, &metrics);
                        }
                        Err(e) => {
                            let event = format!("monitor://error/{sid}");
                            let _ = app.emit(&event, e);
                        }
                    }
                }

                cmd = cmd_rx.recv() => {
                    match cmd {
                        Some(CollectorCommand::SetInterval(ms)) => {
                            interval = time::interval(Duration::from_millis(ms));
                        }
                        Some(CollectorCommand::Stop) | None => break,
                    }
                }
            }
        }
    });

    CollectorState { cmd_tx, task_handle }
}

/// 通过 SSH exec 通道执行单条命令并返回完整输出
async fn exec_command(
    handle: &client::Handle<ClientHandler>,
    command: &str,
) -> Result<String, String> {
    let channel = handle
        .channel_open_session()
        .await
        .map_err(|e| format!("failed to open exec channel: {e}"))?;

    channel
        .exec(true, command)
        .await
        .map_err(|e| format!("exec failed: {e}"))?;

    let mut output = Vec::new();
    loop {
        match channel.wait().await {
            Some(russh::ChannelMsg::Data { data }) => {
                output.extend_from_slice(&data);
            }
            Some(russh::ChannelMsg::Eof) | Some(russh::ChannelMsg::ExitStatus { .. }) => {
                // 继续等待直到 channel 关闭
            }
            None => break,
            _ => {}
        }
    }

    String::from_utf8(output)
        .map_err(|e| format!("invalid UTF-8 in command output: {e}"))
}

/// 检测远程 OS 类型
async fn detect_os(handle: &client::Handle<ClientHandler>) -> Result<ServerOS, String> {
    let output = exec_command(handle, "uname -s").await?;
    let os = output.trim();
    Ok(match os {
        "Linux" => ServerOS::Linux,
        "Darwin" => ServerOS::MacOS,
        "FreeBSD" => ServerOS::FreeBSD,
        _ => ServerOS::Unknown,
    })
}

/// 按分隔符拆分批量命令的输出为各段
fn split_sections(output: &str) -> HashMap<String, String> {
    let mut sections = HashMap::new();
    let mut current_section = String::new();
    let mut current_content = String::new();

    for line in output.lines() {
        if let Some(name) = line.strip_prefix("---").and_then(|s| s.strip_suffix("---")) {
            if !current_section.is_empty() {
                sections.insert(current_section.clone(), current_content.trim().to_string());
            }
            current_section = name.to_string();
            current_content.clear();
        } else {
            current_content.push_str(line);
            current_content.push('\n');
        }
    }
    if !current_section.is_empty() {
        sections.insert(current_section, current_content.trim().to_string());
    }
    sections
}
```

```rust
use std::collections::HashMap;

use super::types::*;

/// 解析 CPU 指标
///
/// Linux: 从 /proc/stat 第一行计算 delta
/// macOS: 从 top -l 输出直接提取百分比
pub fn parse_cpu(
    os: ServerOS,
    raw: &str,
    prev: Option<&RawCpuCounters>,
) -> (CpuMetrics, Option<RawCpuCounters>) {
    match os {
        ServerOS::Linux | ServerOS::FreeBSD => parse_cpu_linux(raw, prev),
        ServerOS::MacOS => (parse_cpu_macos(raw), None),
        ServerOS::Unknown => parse_cpu_linux(raw, prev),
    }
}

fn parse_cpu_linux(raw: &str, prev: Option<&RawCpuCounters>) -> (CpuMetrics, Option<RawCpuCounters>) {
    // 解析: "cpu  user nice system idle iowait irq softirq steal ..."
    let parts: Vec<u64> = raw
        .split_whitespace()
        .skip(1) // 跳过 "cpu" 标签
        .filter_map(|s| s.parse().ok())
        .collect();

    if parts.len() < 8 {
        return (CpuMetrics::default(), None);
    }

    let current = RawCpuCounters {
        user: parts[0],
        nice: parts[1],
        system: parts[2],
        idle: parts[3],
        iowait: parts[4],
        irq: parts[5],
        softirq: parts[6],
        steal: parts[7],
    };

    let cpu = if let Some(prev) = prev {
        let total_delta = current.total().saturating_sub(prev.total());
        if total_delta == 0 {
            CpuMetrics::default()
        } else {
            let idle_delta = current.idle_total().saturating_sub(prev.idle_total());
            let user_delta = (current.user + current.nice).saturating_sub(prev.user + prev.nice);
            let sys_delta = (current.system + current.irq + current.softirq)
                .saturating_sub(prev.system + prev.irq + prev.softirq);
            let iowait_delta = current.iowait.saturating_sub(prev.iowait);

            CpuMetrics {
                usage_percent: 100.0 * (total_delta - idle_delta) as f64 / total_delta as f64,
                user_percent: 100.0 * user_delta as f64 / total_delta as f64,
                system_percent: 100.0 * sys_delta as f64 / total_delta as f64,
                iowait_percent: 100.0 * iowait_delta as f64 / total_delta as f64,
                core_count: 0, // 由 OsInfo 填充
            }
        }
    } else {
        // 首次采样，无法计算 delta，返回零
        CpuMetrics::default()
    };

    (cpu, Some(current))
}

fn parse_cpu_macos(raw: &str) -> CpuMetrics {
    // 解析: "CPU usage: 12.5% user, 8.3% sys, 79.2% idle"
    let mut user = 0.0;
    let mut sys = 0.0;
    let mut idle = 0.0;

    for part in raw.split(',') {
        let part = part.trim();
        if part.contains("user") {
            user = extract_percent(part);
        } else if part.contains("sys") {
            sys = extract_percent(part);
        } else if part.contains("idle") {
            idle = extract_percent(part);
        }
    }

    CpuMetrics {
        usage_percent: 100.0 - idle,
        user_percent: user,
        system_percent: sys,
        iowait_percent: 0.0, // macOS 的 top 不提供 iowait
        core_count: 0,
    }
}

/// 解析 Memory 指标
pub fn parse_memory(os: ServerOS, raw: &str) -> MemoryMetrics {
    match os {
        ServerOS::Linux => parse_memory_linux(raw),
        ServerOS::MacOS => parse_memory_macos(raw),
        _ => parse_memory_linux(raw),
    }
}

fn parse_memory_linux(raw: &str) -> MemoryMetrics {
    let mut fields: HashMap<String, u64> = HashMap::new();

    for line in raw.lines() {
        let parts: Vec<&str> = line.split_whitespace().collect();
        if parts.len() >= 2 {
            let key = parts[0].trim_end_matches(':').to_string();
            if let Ok(val) = parts[1].parse::<u64>() {
                // /proc/meminfo 单位是 kB，转换为 bytes
                fields.insert(key, val * 1024);
            }
        }
    }

    let total = *fields.get("MemTotal").unwrap_or(&0);
    let available = *fields.get("MemAvailable").unwrap_or(&0);
    let swap_total = *fields.get("SwapTotal").unwrap_or(&0);
    let swap_free = *fields.get("SwapFree").unwrap_or(&0);

    let used = total.saturating_sub(available);
    let usage_percent = if total > 0 {
        100.0 * used as f64 / total as f64
    } else {
        0.0
    };

    MemoryMetrics {
        total_bytes: total,
        used_bytes: used,
        available_bytes: available,
        usage_percent,
        swap_total,
        swap_used: swap_total.saturating_sub(swap_free),
    }
}

fn parse_memory_macos(raw: &str) -> MemoryMetrics {
    // vm_stat 输出 + ---MEMSIZE--- 分隔 + hw.memsize
    let page_size: u64 = 4096;
    let mut active: u64 = 0;
    let mut wired: u64 = 0;
    let mut free: u64 = 0;
    let mut total: u64 = 0;

    let mut in_memsize = false;
    for line in raw.lines() {
        if line.contains("---MEMSIZE---") {
            in_memsize = true;
            continue;
        }
        if in_memsize {
            total = line.trim().parse().unwrap_or(0);
            continue;
        }
        if line.contains("Pages active") {
            active = extract_page_count(line);
        } else if line.contains("Pages wired") {
            wired = extract_page_count(line);
        } else if line.contains("Pages free") {
            free = extract_page_count(line);
        }
    }

    let used = (active + wired) * page_size;
    let available = total.saturating_sub(used);
    let usage_percent = if total > 0 {
        100.0 * used as f64 / total as f64
    } else {
        0.0
    };

    MemoryMetrics {
        total_bytes: total,
        used_bytes: used,
        available_bytes: available,
        usage_percent,
        swap_total: 0, // macOS swap 通过 sysctl vm.swapusage 获取，暂不采集
        swap_used: 0,
    }
}

/// 解析 Disk 指标
pub fn parse_disk(os: ServerOS, raw: &str) -> Vec<DiskMetrics> {
    let mut disks = Vec::new();

    for line in raw.lines().skip(1) { // 跳过表头
        let parts: Vec<&str> = line.split_whitespace().collect();
        if parts.len() < 4 {
            continue;
        }

        // Linux df -B1 --output=target,size,used,avail
        // macOS df -b: Filesystem 512-blocks Used Available Capacity Mounted
        let (mount, total, used, avail) = match os {
            ServerOS::MacOS => {
                if parts.len() < 6 { continue; }
                let block_size: u64 = 512;
                (
                    parts[parts.len() - 1].to_string(),
                    parts[1].parse::<u64>().unwrap_or(0) * block_size,
                    parts[2].parse::<u64>().unwrap_or(0) * block_size,
                    parts[3].parse::<u64>().unwrap_or(0) * block_size,
                )
            }
            _ => {
                (
                    parts[0].to_string(),
                    parts[1].parse::<u64>().unwrap_or(0),
                    parts[2].parse::<u64>().unwrap_or(0),
                    parts[3].parse::<u64>().unwrap_or(0),
                )
            }
        };

        let usage_percent = if total > 0 {
            100.0 * used as f64 / total as f64
        } else {
            0.0
        };

        disks.push(DiskMetrics {
            mount_point: mount,
            total_bytes: total,
            used_bytes: used,
            available_bytes: avail,
            usage_percent,
        });
    }

    disks
}

/// 解析 Network 指标（含 delta 速率计算）
pub fn parse_network(
    os: ServerOS,
    raw: &str,
    prev: &HashMap<String, RawNetworkCounters>,
    elapsed_secs: f64,
) -> (NetworkMetrics, HashMap<String, RawNetworkCounters>) {
    let raw_counters = match os {
        ServerOS::Linux => parse_net_linux(raw),
        ServerOS::MacOS => parse_net_macos(raw),
        _ => parse_net_linux(raw),
    };

    let mut interfaces = Vec::new();
    let mut new_counters = HashMap::new();

    for counter in &raw_counters {
        let (rx_rate, tx_rate) = if let Some(prev_counter) = prev.get(&counter.name) {
            let rx_delta = counter.rx_bytes.saturating_sub(prev_counter.rx_bytes);
            let tx_delta = counter.tx_bytes.saturating_sub(prev_counter.tx_bytes);
            if elapsed_secs > 0.0 {
                (rx_delta as f64 / elapsed_secs, tx_delta as f64 / elapsed_secs)
            } else {
                (0.0, 0.0)
            }
        } else {
            (0.0, 0.0)
        };

        interfaces.push(NetworkInterface {
            name: counter.name.clone(),
            rx_bytes_per_sec: rx_rate,
            tx_bytes_per_sec: tx_rate,
            rx_total: counter.rx_bytes,
            tx_total: counter.tx_bytes,
        });

        new_counters.insert(counter.name.clone(), counter.clone());
    }

    (NetworkMetrics { interfaces }, new_counters)
}

/// 解析 Load Average
pub fn parse_load(os: ServerOS, raw: &str) -> LoadAverage {
    match os {
        ServerOS::MacOS => {
            // "{ 0.52 0.38 0.25 }"
            let cleaned = raw.replace(['{', '}'], "");
            let parts: Vec<f64> = cleaned.split_whitespace()
                .filter_map(|s| s.parse().ok())
                .collect();
            LoadAverage {
                one: *parts.first().unwrap_or(&0.0),
                five: *parts.get(1).unwrap_or(&0.0),
                fifteen: *parts.get(2).unwrap_or(&0.0),
            }
        }
        _ => {
            // "0.52 0.38 0.25 1/234 12345"
            let parts: Vec<f64> = raw.split_whitespace()
                .take(3)
                .filter_map(|s| s.parse().ok())
                .collect();
            LoadAverage {
                one: *parts.first().unwrap_or(&0.0),
                five: *parts.get(1).unwrap_or(&0.0),
                fifteen: *parts.get(2).unwrap_or(&0.0),
            }
        }
    }
}

/// 解析 Uptime
pub fn parse_uptime(os: ServerOS, raw: &str) -> f64 {
    match os {
        ServerOS::MacOS => {
            // "{ sec = 1700000000, usec = 0 } ..."
            if let Some(sec_str) = raw.split("sec = ").nth(1) {
                if let Some(sec) = sec_str.split(',').next().and_then(|s| s.trim().parse::<i64>().ok()) {
                    let now = chrono::Utc::now().timestamp();
                    return (now - sec) as f64;
                }
            }
            0.0
        }
        _ => {
            // "123456.78 234567.89"
            raw.split_whitespace()
                .next()
                .and_then(|s| s.parse().ok())
                .unwrap_or(0.0)
        }
    }
}

/// 解析进程列表 (ps aux 输出)
pub fn parse_processes(raw: &str) -> Vec<ProcessInfo> {
    raw.lines()
        .skip(1) // 跳过表头
        .filter_map(|line| {
            let parts: Vec<&str> = line.split_whitespace().collect();
            if parts.len() < 11 {
                return None;
            }
            Some(ProcessInfo {
                pid: parts[1].parse().unwrap_or(0),
                user: parts[0].to_string(),
                cpu_percent: parts[2].parse().unwrap_or(0.0),
                mem_percent: parts[3].parse().unwrap_or(0.0),
                // COMMAND 可能包含空格，取从第 10 列开始的所有内容
                command: parts[10..].join(" "),
            })
        })
        .collect()
}

fn parse_net_linux(raw: &str) -> Vec<RawNetworkCounters> {
    raw.lines()
        .filter_map(|line| {
            let line = line.trim();
            if !line.contains(':') { return None; }

            let (name, rest) = line.split_once(':')?;
            let name = name.trim();

            // 排除 loopback，匹配常见物理/虚拟接口
            if name == "lo" { return None; }

            let parts: Vec<u64> = rest.split_whitespace()
                .filter_map(|s| s.parse().ok())
                .collect();

            if parts.len() < 10 {
                return None;
            }

            Some(RawNetworkCounters {
                name: name.to_string(),
                rx_bytes: parts[0],  // 第 1 列: receive bytes
                tx_bytes: parts[8],  // 第 9 列: transmit bytes
            })
        })
        .collect()
}

fn parse_net_macos(raw: &str) -> Vec<RawNetworkCounters> {
    // netstat -ib 输出:
    // Name  Mtu  Network  Address  Ipkts  Ierrs  Ibytes  Opkts  Oerrs  Obytes  Coll
    raw.lines()
        .filter_map(|line| {
            let parts: Vec<&str> = line.split_whitespace().collect();
            if parts.len() < 11 { return None; }
            let name = parts[0];
            if !name.starts_with("en") { return None; }
            // Ibytes = index 6, Obytes = index 9
            Some(RawNetworkCounters {
                name: name.to_string(),
                rx_bytes: parts[6].parse().unwrap_or(0),
                tx_bytes: parts[9].parse().unwrap_or(0),
            })
        })
        .collect()
}

fn extract_percent(s: &str) -> f64 {
    s.split('%')
        .next()
        .and_then(|p| {
            p.split_whitespace()
                .last()
                .and_then(|n| n.parse().ok())
        })
        .unwrap_or(0.0)
}

fn extract_page_count(line: &str) -> u64 {
    line.split(':')
        .nth(1)
        .and_then(|s| s.trim().trim_end_matches('.').parse().ok())
        .unwrap_or(0)
}
```

```rust
use std::sync::Arc;

use tauri::State;
use tokio::sync::RwLock;

use crate::monitor::collector::{self, CollectorCommand, MetricsHistory};
use crate::monitor::types::SystemMetrics;
use crate::state::AppState;

/// 启动指定会话的监控采集
///
/// 通过已有 SSH 会话的 handle 开启独立的 exec 通道进行指标采集。
/// 如果该会话已在监控中，先停止旧采集器再启动新的。
#[tauri::command]
pub async fn monitor_start(
    state: State<'_, AppState>,
    app: tauri::AppHandle,
    session_id: String,
    interval_ms: Option<u64>,
) -> Result<(), String> {
    let interval = interval_ms.unwrap_or(3000).max(1000); // 最低 1 秒

    // 先停止已有采集
    monitor_stop_inner(&state, &session_id).await;

    // 获取 SSH session 的 handle clone
    let handle = {
        let sessions = state.sessions.read().await;
        let session = sessions
            .get(&session_id)
            .ok_or_else(|| format!("session not found: {session_id}"))?;
        session.handle_clone()
    };

    // 创建历史缓冲区 (5 分钟 / 间隔秒数)
    let capacity = (300_000 / interval) as usize;
    let history = Arc::new(RwLock::new(MetricsHistory::new(capacity.max(10))));

    // 启动采集器
    let collector = collector::spawn_collector(
        handle,
        app,
        session_id.clone(),
        interval,
        history.clone(),
    );

    // 注册到 state
    {
        let mut collectors = state.monitor_collectors.write().await;
        collectors.insert(session_id.clone(), collector);
    }
    {
        let mut histories = state.monitor_history.write().await;
        histories.insert(session_id, history);
    }

    Ok(())
}

/// 停止指定会话的监控采集
#[tauri::command]
pub async fn monitor_stop(
    state: State<'_, AppState>,
    session_id: String,
) -> Result<(), String> {
    monitor_stop_inner(&state, &session_id).await;
    Ok(())
}

async fn monitor_stop_inner(state: &AppState, session_id: &str) {
    // 发送停止命令
    {
        let mut collectors = state.monitor_collectors.write().await;
        if let Some(collector) = collectors.remove(session_id) {
            let _ = collector.cmd_tx.send(CollectorCommand::Stop);
            // 等待任务结束（加超时保护）
            let _ = tokio::time::timeout(
                std::time::Duration::from_secs(5),
                collector.task_handle,
            )
            .await;
        }
    }
    // 清理历史
    {
        let mut histories = state.monitor_history.write().await;
        histories.remove(session_id);
    }
}

/// 获取最新一次指标快照
#[tauri::command]
pub async fn monitor_get_latest(
    state: State<'_, AppState>,
    session_id: String,
) -> Result<Option<SystemMetrics>, String> {
    let histories = state.monitor_history.read().await;
    if let Some(history) = histories.get(&session_id) {
        let hist = history.read().await;
        Ok(hist.latest().cloned())
    } else {
        Ok(None)
    }
}

/// 获取指定时间范围内的历史指标（用于图表渲染）
#[tauri::command]
pub async fn monitor_get_history(
    state: State<'_, AppState>,
    session_id: String,
    duration_secs: Option<u64>,
) -> Result<Vec<SystemMetrics>, String> {
    let duration = duration_secs.unwrap_or(300); // 默认 5 分钟
    let histories = state.monitor_history.read().await;
    if let Some(history) = histories.get(&session_id) {
        let hist = history.read().await;
        Ok(hist.range(duration))
    } else {
        Ok(Vec::new())
    }
}
```

### 5.5 MetricChart 组件：`src/components/monitor/MetricChart.vue`

轻量 Canvas 2D sparkline，不依赖任何图表库：

```vue
<script setup lang="ts">
import { ref, watch, onMounted, onUnmounted } from "vue";

const props = defineProps<{
  /** 数据点数组 (0-100 范围的百分比值) */
  data: number[];
  /** 图表宽度 (px) */
  width?: number;
  /** 图表高度 (px) */
  height?: number;
  /** 线条颜色 (默认根据最新值自动选择) */
  color?: string;
  /** 是否填充面积 */
  fill?: boolean;
}>();

const canvasRef = ref<HTMLCanvasElement | null>(null);
let animationFrame: number | null = null;

/** 根据值获取颜色 */
function getColor(value: number): string {
  if (props.color) return props.color;
  if (value < 60) return "#67c23a";   // 绿色: 正常
  if (value < 85) return "#e6a23c";   // 黄色: 警告
  return "#f56c6c";                    // 红色: 危险
}

function draw() {
  const canvas = canvasRef.value;
  if (!canvas || props.data.length === 0) return;

  const ctx = canvas.getContext("2d");
  if (!ctx) return;

  const w = props.width ?? canvas.clientWidth;
  const h = props.height ?? canvas.clientHeight;
  canvas.width = w * window.devicePixelRatio;
  canvas.height = h * window.devicePixelRatio;
  ctx.scale(window.devicePixelRatio, window.devicePixelRatio);

  ctx.clearRect(0, 0, w, h);

  const data = props.data;
  const step = w / Math.max(data.length - 1, 1);
  const latest = data[data.length - 1] ?? 0;
  const color = getColor(latest);

  // 绘制面积填充
  if (props.fill !== false) {
    ctx.beginPath();
    ctx.moveTo(0, h);
    data.forEach((v, i) => {
      const x = i * step;
      const y = h - (v / 100) * h;
      ctx.lineTo(x, y);
    });
    ctx.lineTo((data.length - 1) * step, h);
    ctx.closePath();
    ctx.fillStyle = color + "20"; // 12.5% 透明度
    ctx.fill();
  }

  // 绘制线条
  ctx.beginPath();
  data.forEach((v, i) => {
    const x = i * step;
    const y = h - (v / 100) * h;
    if (i === 0) ctx.moveTo(x, y);
    else ctx.lineTo(x, y);
  });
  ctx.strokeStyle = color;
  ctx.lineWidth = 1.5;
  ctx.stroke();
}

watch(() => props.data, () => {
  if (animationFrame) cancelAnimationFrame(animationFrame);
  animationFrame = requestAnimationFrame(draw);
}, { deep: true });

onMounted(draw);
onUnmounted(() => {
  if (animationFrame) cancelAnimationFrame(animationFrame);
});
</script>

<template>
  <canvas
    ref="canvasRef"
    :style="{ width: `${width ?? 200}px`, height: `${height ?? 40}px` }"
    class="metric-chart"
  />
</template>

<style scoped>
.metric-chart {
  display: block;
}
</style>
```

### 6.1 Monitor Store：`src/stores/monitorStore.ts`

```typescript
import { defineStore } from "pinia";
import { ref, reactive } from "vue";
import { listen } from "@tauri-apps/api/event";
import type { SystemMetrics, OsInfo } from "@/types/monitor";
import { tauriInvoke } from "@/utils/tauri";

export const useMonitorStore = defineStore("monitor", () => {
  /** 每个 session 的最新指标 */
  const latestMetrics = reactive<Record<string, SystemMetrics>>({});

  /** 每个 session 的历史指标 (用于图表) */
  const metricsHistory = reactive<Record<string, SystemMetrics[]>>({});

  /** 每个 session 的 OS 信息 */
  const osInfoMap = reactive<Record<string, OsInfo>>({});

  /** 正在采集的 session 集合 */
  const collectingSessions = ref<Set<string>>(new Set());

  /** 事件监听清理函数 */
  const unlistenMap = new Map<string, (() => void)[]>();

  /** 历史记录最大长度 (前端侧) */
  const MAX_HISTORY = 100;

  /** 启动监控 */
  async function start(sessionId: string, intervalMs?: number) {
    await tauriInvoke("monitor_start", {
      sessionId,
      intervalMs: intervalMs ?? 3000,
    });

    collectingSessions.value.add(sessionId);

    // 监听指标事件
    const unlistenMetrics = await listen<SystemMetrics>(
      `monitor://metrics/${sessionId}`,
      (event) => {
        const metrics = event.payload;
        latestMetrics[sessionId] = metrics;

        // 追加到历史
        if (!metricsHistory[sessionId]) {
          metricsHistory[sessionId] = [];
        }
        metricsHistory[sessionId].push(metrics);
        if (metricsHistory[sessionId].length > MAX_HISTORY) {
          metricsHistory[sessionId].shift();
        }
      }
    );

    // 监听 OS 信息事件
    const unlistenOs = await listen<OsInfo>(
      `monitor://os-info/${sessionId}`,
      (event) => {
        osInfoMap[sessionId] = event.payload;
      }
    );

    // 监听错误事件
    const unlistenError = await listen<string>(
      `monitor://error/${sessionId}`,
      (event) => {
        console.warn(`[monitor] collection error for ${sessionId}:`, event.payload);
      }
    );

    unlistenMap.set(sessionId, [unlistenMetrics, unlistenOs, unlistenError]);
  }

  /** 停止监控 */
  async function stop(sessionId: string) {
    await tauriInvoke("monitor_stop", { sessionId });
    collectingSessions.value.delete(sessionId);

    // 清理事件监听
    const unlisteners = unlistenMap.get(sessionId);
    if (unlisteners) {
      unlisteners.forEach((fn) => fn());
      unlistenMap.delete(sessionId);
    }
  }

  /** 清理指定 session 的所有状态（断开连接时调用） */
  function cleanup(sessionId: string) {
    collectingSessions.value.delete(sessionId);
    delete latestMetrics[sessionId];
    delete metricsHistory[sessionId];
    delete osInfoMap[sessionId];
    const unlisteners = unlistenMap.get(sessionId);
    if (unlisteners) {
      unlisteners.forEach((fn) => fn());
      unlistenMap.delete(sessionId);
    }
  }

  /** 获取最新指标 */
  function getLatest(sessionId: string): SystemMetrics | undefined {
    return latestMetrics[sessionId];
  }

  /** 获取历史指标 */
  function getHistory(sessionId: string): SystemMetrics[] {
    return metricsHistory[sessionId] ?? [];
  }

  /** 获取 OS 信息 */
  function getOsInfo(sessionId: string): OsInfo | undefined {
    return osInfoMap[sessionId];
  }

  /** 是否正在采集 */
  function isCollecting(sessionId: string): boolean {
    return collectingSessions.value.has(sessionId);
  }

  return {
    latestMetrics,
    metricsHistory,
    osInfoMap,
    collectingSessions,
    start,
    stop,
    cleanup,
    getLatest,
    getHistory,
    getOsInfo,
    isCollecting,
  };
});
```

### 9.1 字节格式化

**新增函数**: `src/utils/format.ts`

```typescript
/** 格式化字节数为人类可读格式 */
export function formatBytes(bytes: number, decimals = 1): string {
  if (bytes === 0) return "0 B";
  const k = 1024;
  const sizes = ["B", "KB", "MB", "GB", "TB"];
  const i = Math.floor(Math.log(bytes) / Math.log(k));
  return `${(bytes / Math.pow(k, i)).toFixed(decimals)} ${sizes[i]}`;
}

/** 格式化字节速率 */
export function formatBytesPerSec(bytesPerSec: number): string {
  return `${formatBytes(bytesPerSec)}/s`;
}

/** 格式化 uptime 秒数为人类可读格式 */
export function formatUptime(seconds: number): string {
  const days = Math.floor(seconds / 86400);
  const hours = Math.floor((seconds % 86400) / 3600);
  const mins = Math.floor((seconds % 3600) / 60);

  if (days > 0) return `${days}d ${hours}h`;
  if (hours > 0) return `${hours}h ${mins}m`;
  return `${mins}m`;
}
```

## 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| SSH exec 通道开销过大 | 增加服务器负载 | 批量命令合并为单次 exec，默认 3s 间隔 |
| 服务器兼容性差异 | 部分命令不存在 | 每段独立 try/catch，失败显示 N/A |
| 受限 shell (rbash) | 命令被禁止执行 | 检测到 exec 失败时整体降级，提示用户 |
| 网络延迟导致指标积压 | 图表跳跃 | exec 超时设为 interval_ms - 500，超时放弃本轮 |
| busybox 环境 | ps/df 参数不兼容 | 检测 busybox 后使用兼容参数 (`df -k` 代替 `df -B1`) |
| 内存占用 | 历史数据增长 | 环形缓冲区固定容量，5 分钟自动淘汰 |
| Canvas 渲染性能 | 低端设备卡顿 | 使用 requestAnimationFrame 节流，数据点限制 100 个 |
