# v0.32.0 — Terminal Split Pane + Broadcast: 终端分屏与广播输入

> 状态：**计划中**
> 依赖：v0.31.0

---

## 背景

### 当前终端架构

Termex 的终端采用 **1 tab = 1 session = 1 terminal** 的扁平架构：

```
TerminalTabs.vue
  └── TerminalPane.vue (per tab)
        └── TabWorkspace.vue
              ├── TerminalView.vue (xterm.js instance)
              └── SftpPanel.vue (optional split)
```

每个 `Tab` 绑定一个 `sessionId`，对应一个 SSH 连接。`TabWorkspace` 管理终端与 SFTP 面板之间的分屏。

**核心限制**:
1. **单终端视图** — 无法在一个 tab 内打开多个终端 (对比 iTerm2 / tmux 分屏)
2. **无广播输入** — 无法同时向多个服务器发送相同命令 (对比 csshX / tmux synchronize-panes)
3. **多服务器操作低效** — 需要在多个 tab 之间切换，无法并排对比输出

### 目标

实现 iTerm2 风格的原生终端分屏 + tmux 风格的广播输入，不依赖远程 tmux。

### 设计决策

- **纯前端实现** — 分屏是 UI 布局概念，不涉及后端改动。每个 pane 仍是独立的 SSH session
- **二叉树布局** — 每次分屏将一个 pane 分为两个，形成递归二叉树结构
- **与现有 SFTP 分屏共存** — `TabWorkspace` 的 SFTP 分屏功能保留，pane 分屏在更高层级
- **不侵入 xterm.js** — 每个 pane 仍是独立的 `TerminalView` 组件，分屏由外层容器管理

## 一、Pane Layout 数据模型

### 1.1 类型定义

**新建文件**: `src/types/paneLayout.ts`

```typescript
/** A node in the pane binary tree — either a leaf (terminal) or a split (container). */
export type PaneNode = PaneLeaf | PaneSplit;

/** Leaf node: contains a single terminal pane. */
export interface PaneLeaf {
  type: "leaf";
  /** Unique pane identifier. */
  id: string;
  /** SSH session ID for this pane. */
  sessionId: string;
  /** Server ID this pane connects to (for reconnect / clone). */
  serverId: string;
  /** Display title (server name or custom label). */
  title: string;
}

/** Split node: contains two children with a divider. */
export interface PaneSplit {
  type: "split";
  /** Unique split node identifier. */
  id: string;
  /** Split direction. */
  direction: "horizontal" | "vertical";
  /** Divider position as ratio (0.0 - 1.0). Default 0.5. */
  ratio: number;
  /** Exactly two children: [first, second].
   *  horizontal: first=top, second=bottom.
   *  vertical: first=left, second=right. */
  children: [PaneNode, PaneNode];
}

/** Broadcast mode state for a tab. */
export interface BroadcastState {
  /** Whether broadcast mode is active. */
  enabled: boolean;
  /** Pane IDs that receive broadcast input.
   *  When empty + enabled, ALL panes receive input. */
  includedPaneIds: Set<string>;
}

/** Minimum pane dimensions (in CSS pixels). */
export const MIN_PANE_WIDTH = 200;
export const MIN_PANE_HEIGHT = 120;

/** Minimum ratio to prevent panes from collapsing. */
export const MIN_SPLIT_RATIO = 0.15;
export const MAX_SPLIT_RATIO = 0.85;
```

### 1.2 Tree 工具函数

**新建文件**: `src/utils/paneTree.ts` (~200 行)

纯函数工具集，操作 `PaneNode` 二叉树。

```typescript
import { v4 as uuid } from "uuid";
import type { PaneNode, PaneLeaf, PaneSplit } from "@/types/paneLayout";

/** Creates a new leaf pane. */
export function createLeaf(sessionId: string, serverId: string, title: string): PaneLeaf {
  return {
    type: "leaf",
    id: uuid(),
    sessionId,
    serverId,
    title,
  };
}

/** Splits a target pane into two, returning a new tree root. */
export function splitPane(
  root: PaneNode,
  targetPaneId: string,
  direction: "horizontal" | "vertical",
  newLeaf: PaneLeaf,
): PaneNode {
  return transformNode(root, targetPaneId, (leaf) => ({
    type: "split",
    id: uuid(),
    direction,
    ratio: 0.5,
    children: [leaf, newLeaf],
  }));
}

/** Closes a pane and collapses its parent split. */
export function closePane(root: PaneNode, targetPaneId: string): PaneNode | null {
  if (root.type === "leaf") {
    return root.id === targetPaneId ? null : root;
  }

  const [first, second] = root.children;

  // If target is a direct child, return the sibling
  if (first.type === "leaf" && first.id === targetPaneId) return second;
  if (second.type === "leaf" && second.id === targetPaneId) return first;

  // Recurse into children
  const newFirst = closePane(first, targetPaneId);
  const newSecond = closePane(second, targetPaneId);

  if (newFirst === null) return newSecond;
  if (newSecond === null) return newFirst;

  return { ...root, children: [newFirst, newSecond] };
}

/** Updates the split ratio for a specific split node. */
export function updateRatio(root: PaneNode, splitId: string, ratio: number): PaneNode {
  const clamped = Math.max(MIN_SPLIT_RATIO, Math.min(MAX_SPLIT_RATIO, ratio));
  return transformNode(root, splitId, (node) => {
    if (node.type === "split") {
      return { ...node, ratio: clamped };
    }
    return node;
  });
}

/** Collects all leaf pane IDs in DFS order. */
export function collectLeafIds(node: PaneNode): string[] {
  if (node.type === "leaf") return [node.id];
  return [
    ...collectLeafIds(node.children[0]),
    ...collectLeafIds(node.children[1]),
  ];
}

/** Finds a leaf by ID. */
export function findLeaf(node: PaneNode, id: string): PaneLeaf | null {
  if (node.type === "leaf") return node.id === id ? node : null;
  return findLeaf(node.children[0], id) ?? findLeaf(node.children[1], id);
}

/** Counts total leaf panes. */
export function countLeaves(node: PaneNode): number {
  if (node.type === "leaf") return 1;
  return countLeaves(node.children[0]) + countLeaves(node.children[1]);
}

/** Finds the next/previous leaf in DFS order for focus navigation. */
export function findAdjacentLeaf(
  root: PaneNode,
  currentId: string,
  direction: "next" | "prev",
): string | null {
  const ids = collectLeafIds(root);
  const idx = ids.indexOf(currentId);
  if (idx === -1) return null;

  if (direction === "next") {
    return ids[(idx + 1) % ids.length];
  } else {
    return ids[(idx - 1 + ids.length) % ids.length];
  }
}

/** Internal: recursively transforms a node matching targetId. */
function transformNode(
  node: PaneNode,
  targetId: string,
  transform: (node: PaneNode) => PaneNode,
): PaneNode {
  if (node.type === "leaf" && node.id === targetId) {
    return transform(node);
  }
  if (node.type === "split") {
    if (node.id === targetId) return transform(node);
    return {
      ...node,
      children: [
        transformNode(node.children[0], targetId, transform),
        transformNode(node.children[1], targetId, transform),
      ],
    };
  }
  return node;
}
```

**修改文件**: `src/stores/sessionStore.ts`

```typescript
import type { PaneNode, PaneLeaf, BroadcastState } from "@/types/paneLayout";
import * as paneTree from "@/utils/paneTree";

// 新增 state
/** Pane layout tree per tab, keyed by tabKey. */
const paneLayouts = ref<Map<string, PaneNode>>(new Map());
/** Currently focused pane ID. */
const activePaneId = ref<string | null>(null);
/** Broadcast state per tab, keyed by tabKey. */
const broadcastStates = ref<Map<string, BroadcastState>>(new Map());

// 修改 connect() — 创建 tab 时同时初始化单 pane 布局
async function connect(serverId: string, serverName: string): Promise<void> {
  // ... 现有逻辑 ...

  // After tab creation, initialize pane layout
  const initialLeaf = paneTree.createLeaf(placeholderId, serverId, serverName);
  paneLayouts.value.set(tabKey, initialLeaf);
  activePaneId.value = initialLeaf.id;

  // ... SSH connect logic ...

  // After successful connect, update pane's sessionId
  updatePaneSessionId(tabKey, initialLeaf.id, realId);
}

/** Updates a pane's sessionId (after SSH connect succeeds). */
function updatePaneSessionId(tabKey: string, paneId: string, newSessionId: string): void {
  const layout = paneLayouts.value.get(tabKey);
  if (!layout) return;
  const leaf = paneTree.findLeaf(layout, paneId);
  if (leaf) {
    leaf.sessionId = newSessionId;
  }
}

/** Splits the active pane and connects the new pane to the same server. */
async function splitActivePane(
  direction: "horizontal" | "vertical",
): Promise<void> {
  const currentTab = activeTab.value;
  if (!currentTab || !activePaneId.value) return;

  const layout = paneLayouts.value.get(currentTab.tabKey);
  if (!layout) return;

  const currentLeaf = paneTree.findLeaf(layout, activePaneId.value);
  if (!currentLeaf) return;

  // Create new pane targeting same server
  const newLeaf = paneTree.createLeaf(
    `connecting-${Date.now()}`,
    currentLeaf.serverId,
    currentLeaf.title,
  );

  // Update tree
  const newLayout = paneTree.splitPane(layout, activePaneId.value, direction, newLeaf);
  paneLayouts.value.set(currentTab.tabKey, newLayout);

  // Connect new pane to SSH
  try {
    const realId = await tauriInvoke<string>("ssh_connect", {
      serverId: currentLeaf.serverId,
    });
    updatePaneSessionId(currentTab.tabKey, newLeaf.id, realId);

    // Register session
    const session: Session = {
      id: realId,
      serverId: currentLeaf.serverId,
      serverName: currentLeaf.title,
      status: "authenticated",
      startedAt: new Date().toISOString(),
      type: "ssh",
    };
    sessions.value.set(realId, session);
  } catch (err) {
    // Remove new pane on failure
    const reverted = paneTree.closePane(newLayout, newLeaf.id);
    if (reverted) {
      paneLayouts.value.set(currentTab.tabKey, reverted);
    }
  }
}

/** Closes a specific pane. If it's the last pane, closes the tab. */
function closePane(paneId: string): void {
  const currentTab = activeTab.value;
  if (!currentTab) return;

  const layout = paneLayouts.value.get(currentTab.tabKey);
  if (!layout) return;

  const leaf = paneTree.findLeaf(layout, paneId);
  if (!leaf) return;

  // Disconnect the session
  if (leaf.sessionId && !leaf.sessionId.startsWith("connecting-")) {
    tauriInvoke("ssh_disconnect", { sessionId: leaf.sessionId }).catch(() => {});
    sessions.value.delete(leaf.sessionId);
  }

  const newLayout = paneTree.closePane(layout, paneId);
  if (!newLayout) {
    // Last pane — close the entire tab
    closeTab(currentTab.tabKey);
    return;
  }

  paneLayouts.value.set(currentTab.tabKey, newLayout);

  // Move focus to sibling
  if (activePaneId.value === paneId) {
    const leaves = paneTree.collectLeafIds(newLayout);
    activePaneId.value = leaves[0] ?? null;
  }
}

/** Toggles broadcast mode for the current tab. */
function toggleBroadcast(): void {
  const currentTab = activeTab.value;
  if (!currentTab) return;

  const state = broadcastStates.value.get(currentTab.tabKey) ?? {
    enabled: false,
    includedPaneIds: new Set<string>(),
  };

  state.enabled = !state.enabled;

  // When enabling, include all panes by default
  if (state.enabled) {
    const layout = paneLayouts.value.get(currentTab.tabKey);
    if (layout) {
      state.includedPaneIds = new Set(paneTree.collectLeafIds(layout));
    }
  }

  broadcastStates.value.set(currentTab.tabKey, state);
}

/** Toggles a specific pane's inclusion in broadcast. */
function togglePaneBroadcast(paneId: string): void {
  const currentTab = activeTab.value;
  if (!currentTab) return;

  const state = broadcastStates.value.get(currentTab.tabKey);
  if (!state) return;

  if (state.includedPaneIds.has(paneId)) {
    state.includedPaneIds.delete(paneId);
  } else {
    state.includedPaneIds.add(paneId);
  }
}

// 新增 getters
const currentPaneLayout = computed(() => {
  const tab = activeTab.value;
  return tab ? paneLayouts.value.get(tab.tabKey) ?? null : null;
});

const currentBroadcast = computed(() => {
  const tab = activeTab.value;
  return tab ? broadcastStates.value.get(tab.tabKey) ?? null : null;
});

const paneCount = computed(() => {
  const layout = currentPaneLayout.value;
  return layout ? paneTree.countLeaves(layout) : 0;
});
```

<script setup lang="ts">
import { ref } from "vue";
import { useSessionStore } from "@/stores/sessionStore";
import { MIN_SPLIT_RATIO, MAX_SPLIT_RATIO } from "@/types/paneLayout";
import * as paneTree from "@/utils/paneTree";

const props = defineProps<{
  splitId: string;
  direction: "horizontal" | "vertical";
  ratio: number;
}>();

const sessionStore = useSessionStore();
const isDragging = ref(false);

function onMouseDown(e: MouseEvent) {
  e.preventDefault();
  isDragging.value = true;

  const startPos = props.direction === "horizontal" ? e.clientY : e.clientX;
  const startRatio = props.ratio;

  // Get parent container dimensions
  const container = (e.target as HTMLElement).parentElement;
  if (!container) return;
  const containerSize = props.direction === "horizontal"
    ? container.clientHeight
    : container.clientWidth;

  function onMouseMove(ev: MouseEvent) {
    const currentPos = props.direction === "horizontal" ? ev.clientY : ev.clientX;
    const delta = (currentPos - startPos) / containerSize;
    const newRatio = Math.max(MIN_SPLIT_RATIO, Math.min(MAX_SPLIT_RATIO, startRatio + delta));

    // Update layout in store
    const tab = sessionStore.activeTab;
    if (!tab) return;
    const layout = sessionStore.paneLayouts.get(tab.tabKey);
    if (!layout) return;
    const updated = paneTree.updateRatio(layout, props.splitId, newRatio);
    sessionStore.paneLayouts.set(tab.tabKey, updated);
  }

  function onMouseUp() {
    isDragging.value = false;
    document.removeEventListener("mousemove", onMouseMove);
    document.removeEventListener("mouseup", onMouseUp);

    // Trigger terminal resize on all panes
    // (handled by ResizeObserver in each TerminalView)
  }

  document.addEventListener("mousemove", onMouseMove);
  document.addEventListener("mouseup", onMouseUp);
}
</script>

<template>
  <div
    class="pane-divider shrink-0 transition-colors"
    :class="[
      direction === 'horizontal'
        ? 'w-full h-1 cursor-row-resize hover:bg-blue-500/30'
        : 'h-full w-1 cursor-col-resize hover:bg-blue-500/30',
      isDragging ? 'bg-blue-500/50' : 'bg-transparent',
    ]"
    style="z-index: 10"
    @mousedown="onMouseDown"

</template>

## 四、Broadcast Input: 广播输入

### 4.1 Broadcast Composable

**新建文件**: `src/composables/useBroadcast.ts` (~100 行)

拦截键盘输入并路由到多个 session。

```typescript
import { computed, watch } from "vue";
import { useSessionStore } from "@/stores/sessionStore";
import { tauriInvoke } from "@/utils/tauri";
import * as paneTree from "@/utils/paneTree";

export function useBroadcast() {
  const sessionStore = useSessionStore();

  const isActive = computed(() => sessionStore.currentBroadcast?.enabled ?? false);
  const broadcastPaneIds = computed(() => sessionStore.currentBroadcast?.includedPaneIds ?? new Set());

  /** Resolves session IDs for all broadcast-included panes. */
  function getBroadcastSessionIds(): string[] {
    const tab = sessionStore.activeTab;
    if (!tab) return [];

    const layout = sessionStore.paneLayouts.get(tab.tabKey);
    if (!layout) return [];

    const allLeafIds = paneTree.collectLeafIds(layout);
    const includedIds = broadcastPaneIds.value;

    return allLeafIds
      .filter(id => includedIds.has(id))
      .map(id => {
        const leaf = paneTree.findLeaf(layout, id);
        return leaf?.sessionId ?? null;
      })
      .filter((id): id is string => id !== null && !id.startsWith("connecting-"));
  }

  /** Sends input data to all broadcast targets.
   *  Called from useTerminal.ts when broadcast is active. */
  async function broadcastInput(data: string): Promise<void> {
    if (!isActive.value) return;

    const sessionIds = getBroadcastSessionIds();
    const activeSid = sessionStore.activeSession?.id;

    // Send to all broadcast targets EXCEPT the active one (already handled by normal input)
    const targets = sessionIds.filter(sid => sid !== activeSid);

    await Promise.allSettled(
      targets.map(sid => tauriInvoke("ssh_write", { sessionId: sid, data }))
    );
  }

  return {
    isActive,
    broadcastPaneIds,
    broadcastInput,
    getBroadcastSessionIds,
  };
}
```

### 5.4 Directional Focus Navigation

方向键焦点导航需要理解 pane 的空间位置。使用二叉树路径推导:

```typescript
/** Finds the pane in a given direction relative to the current pane.
 *  Uses the split tree structure to determine spatial relationships. */
export function findPaneInDirection(
  root: PaneNode,
  currentId: string,
  direction: "up" | "down" | "left" | "right",
): string | null {
  // Strategy: walk up the tree from current leaf until finding a split
  // whose direction matches the movement, then descend into the opposite child.
  //
  // "up" → find nearest horizontal split where current is in bottom child → go to top
  // "down" → find nearest horizontal split where current is in top child → go to bottom
  // "left" → find nearest vertical split where current is in right child → go to left
  // "right" → find nearest vertical split where current is in left child → go to right

  const path = findPathToLeaf(root, currentId);
  if (!path) return null;

  const targetAxis = (direction === "up" || direction === "down") ? "horizontal" : "vertical";
  const targetChild = (direction === "up" || direction === "left") ? 1 : 0;
  // targetChild=1 means current is in children[1] and we want children[0]

  // Walk up the path looking for matching split
  for (let i = path.length - 1; i >= 0; i--) {
    const { node, childIndex } = path[i];
    if (node.type === "split" && node.direction === targetAxis && childIndex === targetChild) {
      // Found! Descend into the other child to find the nearest leaf
      const otherChild = node.children[1 - targetChild];
      return findNearestLeaf(otherChild, direction);
    }
  }

  return null;  // No pane in that direction
}
```

## 六、Terminal Resize Handling

### 6.1 ResizeObserver Integration

每个 `TerminalView` 已经使用 `ResizeObserver` 来 fit terminal。分屏时:

1. 用户拖拽 `PaneDivider` → `ratio` 更新 → CSS flex/height 变化
2. 浏览器重新布局 → `ResizeObserver` 在每个受影响 pane 上触发
3. `TerminalView.fit()` → `xterm.js` `FitAddon.fit()` → `ssh_resize` IPC

**无需额外代码** — 现有 `ResizeObserver` 机制自动处理。

### 6.2 Minimum Size Enforcement

在 `PaneDivider` 拖拽逻辑中，`MIN_SPLIT_RATIO` (0.15) 确保每个 pane 至少占父容器的 15%。在标准 1440px 宽的窗口中，最小 pane 宽度约 216px（足以容纳 ~30 列终端字符）。

深度嵌套时 (3+ 层分屏)，单 pane 可能变得非常小。限制:

```typescript
/** Maximum split depth. Prevents infinitely small panes. */
export const MAX_SPLIT_DEPTH = 4;  // 允许最多 16 个 pane

export function getTreeDepth(node: PaneNode): number {
  if (node.type === "leaf") return 0;
  return 1 + Math.max(
    getTreeDepth(node.children[0]),
    getTreeDepth(node.children[1]),
  );
}

// In splitActivePane():
if (getTreeDepth(layout) >= MAX_SPLIT_DEPTH) {
  ElMessage.warning("Maximum split depth reached");
  return;
}
```

### 7.2 Tab Status Aggregation

Tab 状态需要聚合所有 pane 的状态：

```typescript
/** Aggregate tab status from all pane sessions. */
const tabStatus = computed(() => {
  const tab = activeTab.value;
  if (!tab) return "disconnected";
  const layout = paneLayouts.value.get(tab.tabKey);
  if (!layout) return "disconnected";

  const leafIds = paneTree.collectLeafIds(layout);
  const statuses = leafIds
    .map(id => {
      const leaf = paneTree.findLeaf(layout, id);
      if (!leaf) return "disconnected";
      const session = sessions.value.get(leaf.sessionId);
      return session?.status ?? "disconnected";
    });

  // If any pane is connected, tab is connected
  if (statuses.some(s => s === "connected")) return "connected";
  if (statuses.some(s => s === "authenticated")) return "authenticated";
  if (statuses.some(s => s === "connecting")) return "connecting";
  if (statuses.some(s => s === "error")) return "error";
  return "disconnected";
});
```

## 十、已知限制与后续迭代

1. **Pane 布局不持久化** — 当前分屏布局存内存，关闭 tab / 应用后丢失。v0.33 考虑持久化到 session 恢复机制
2. **同服务器分屏** — 默认新 pane 连接到相同服务器。"连接到其他服务器" 需要弹出 server picker，本版本未实现
3. **Pane 拖拽交换** — 拖拽 pane header 将 pane 移到另一个位置 (swap)，本版本不实现，计划 v0.34
4. **跨 tab 分屏** — 将一个 tab 的 session 拖到另一个 tab 合并为分屏，不在本版本范围
5. **终端字体独立设置** — 所有 pane 共用全局字体设置，不支持 per-pane 字体
6. **Local PTY 分屏** — 本版本分屏仅支持 SSH session，本地终端 (local PTY) 的分屏待后续适配
7. **Split direction detection** — 方向键导航的空间感知基于树结构推导，非像素级碰撞检测。极端不均匀分屏时可能导航不直观
