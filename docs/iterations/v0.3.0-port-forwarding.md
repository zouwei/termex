# v0.3.0 -- Port Forwarding + Config Export/Import

> Status: **Planned**
> Depends on: v0.1.0

## Goals

实现 SSH 端口转发三种模式，以及加密配置导入/导出功能，提升运维效率和配置可迁移性。

## Features

### Port Forwarding
- [ ] 本地端口转发（Local Forward: -L）
- [ ] 远程端口转发（Remote Forward: -R）
- [ ] 动态端口转发（Dynamic/SOCKS: -D）
- [ ] 转发规则 CRUD（关联服务器，支持自动启动）
- [ ] 转发状态管理（启动/停止/错误检测）
- [ ] 端口转发面板 UI
- [ ] commands/port_forward.rs + 单元测试

### Config Export/Import
- [ ] .termex 导出文件格式实现
  - Magic bytes "TMEX" (4B) + Version (2B) + Salt (16B) + Encrypted Payload + Auth Tag (16B)
  - Payload: JSON { servers, groups, settings, keys, port_forwards } → gzip → AES-256-GCM
- [ ] 独立导出密码（与主密码解耦）
- [ ] 导出选项（选择性导出：服务器/凭证/设置/转发规则）
- [ ] 导入冲突处理（跳过/覆盖/重命名）
- [ ] commands/crypto.rs 扩展 config_export / config_import
- [ ] 前端导出/导入对话框
- [ ] 加密导出/导入单元测试

### IPC Commands
- port_forward_start, stop, list, save, delete
- config_export, config_import
