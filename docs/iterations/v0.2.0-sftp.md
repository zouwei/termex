# v0.2.0 -- SFTP File Browser

> Status: **Planned**
> Depends on: v0.1.0

## Goals

基于已有 SSH 连接，实现 SFTP 文件浏览与传输，提供双栏文件管理器体验。

## Features

### Backend (Rust)
- [ ] SFTP 子系统初始化（复用 SSH session 的 russh channel）
- [ ] 目录列表（list_dir）
- [ ] 文件上传（upload，支持断点续传）
- [ ] 文件下载（download）
- [ ] 文件/目录删除（delete）
- [ ] 重命名（rename）
- [ ] 创建目录（mkdir）
- [ ] 文件内容读取（read_file，用于在线编辑小文件）
- [ ] 传输进度事件推送（sftp://progress, sftp://complete）
- [ ] SFTP commands 模块 + 单元测试

### Frontend (Vue)
- [ ] SftpPanel.vue -- SFTP 面板容器（底部折叠面板）
- [ ] FileList.vue -- 双栏文件列表（本地 + 远程）
- [ ] TransferBar.vue -- 传输队列 + 进度条
- [ ] 文件图标识别（按扩展名）
- [ ] 拖拽上传/下载
- [ ] 右键菜单（下载/上传/删除/重命名/新建目录）
- [ ] 文件大小/权限/修改时间显示

### IPC Commands
- sftp_list_dir
- sftp_upload
- sftp_download
- sftp_delete
- sftp_rename
- sftp_mkdir
- sftp_read_file
