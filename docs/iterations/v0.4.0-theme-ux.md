# v0.4.0 -- Theme System + Settings Persistence + UX Polish

> Status: **Planned**
> Depends on: v0.1.0

## Goals

完善用户体验：主题系统、设置持久化、快捷键可配置，以及整体 UI 打磨。

## Features

### Theme System
- [ ] Dark / Light / System 三种模式
- [ ] CSS 变量驱动的主题切换
- [ ] 终端配色方案（预设 5+ 方案：Termex Dark, Solarized, Monokai, Nord, Dracula）
- [ ] 自定义终端配色编辑器
- [ ] 窗口透明度调节

### Settings Persistence
- [ ] commands/settings.rs -- settings_get, settings_set, settings_get_all
- [ ] settingsStore 加载/保存到 SQLCipher settings 表
- [ ] 所有设置项持久化（主题、字体、光标、语言、面板可见性等）
- [ ] 设置变更实时生效（watch + apply）

### Keybindings
- [ ] KeybindingsTab.vue -- 快捷键查看 + 自定义
- [ ] 快捷键冲突检测
- [ ] 快捷键重置为默认

### UX Polish
- [ ] 侧边栏宽度可拖拽调整
- [ ] 右键上下文菜单（服务器节点：连接/编辑/删除/复制）
- [ ] 服务器拖拽排序 + 拖拽到分组
- [ ] 连接失败重试机制
- [ ] 断线重连提示
- [ ] 窗口标题显示当前连接信息
- [ ] SecurityTab.vue -- 主密码管理 UI
- [ ] AboutTab.vue -- 版本信息 + 开源协议

### Tests
- [ ] 设置持久化往返测试
- [ ] 主题切换测试
- [ ] 快捷键注册/冲突测试
