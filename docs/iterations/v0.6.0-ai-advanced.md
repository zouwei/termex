# v0.6.0 -- AI Advanced: NL2Cmd + Smart Autocomplete

> Status: **Planned**
> Depends on: v0.5.0

## Goals

AI 能力从被动防御升级到主动辅助：自然语言转命令 + 基于上下文的智能补全。

## Features

### Natural Language to Command (NL2Cmd)
- [ ] AiPanel.vue -- AI 对话面板（右侧折叠面板）
- [ ] AiInput.vue -- 自然语言输入框
- [ ] AiMessage.vue -- 消息气泡（用户 / AI）
- [ ] 输入 "查看内存占用最高的10个进程" → 生成 `ps aux --sort=-%mem | head -11`
- [ ] 生成命令一键插入终端（不自动执行，用户确认后发送）
- [ ] 会话上下文感知（当前目录、OS 类型、已安装软件）
- [ ] aiStore -- 消息管理、流式状态

### Smart Autocomplete
- [ ] 基于当前服务器环境的命令补全建议
- [ ] 命令历史智能搜索（自然语言搜索历史命令）
- [ ] 补全面板 UI（终端内嵌入式弹出）

### Multi-Provider Enhancement
- [ ] Provider 健康检查 + 自动切换
- [ ] 请求超时 + 重试策略
- [ ] Token 用量统计显示
- [ ] Custom Provider 支持（自定义 OpenAI-compatible endpoint）
