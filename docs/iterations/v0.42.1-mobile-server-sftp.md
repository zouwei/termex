# v0.42.1 — Mobile Server Management + SFTP + Settings

> 状态：**计划中**
> 依赖：v0.40.0 (移动端基础)
> 关联：移动端路线图第 2 迭代（与跨设备同步合并）

---

## 背景

v0.40.0 完成了移动端最小 SSH 客户端，用户可以在手机上连接服务器并使用终端。本迭代补齐移动端的服务器管理、SFTP 文件操作和基础设置功能。

### 目标

1. 移动端完整服务器 CRUD（新建、编辑、删除、分组）
2. 移动端 SFTP 文件浏览、上传、下载
3. 移动端基础设置（终端外观、语言、AI 供应商）
4. 系统 API 移动端适配（剪贴板、文件选择器、URL 打开）

> **iPad**: 本迭代仍走手机布局，iPad 侧边栏布局推迟至 v0.46.0+。

---

## 一、移动端服务器管理

### 1.1 MobileConnectForm.vue

**新建文件**: `src/components/mobile/MobileConnectForm.vue`

桌面端 `ConnectModal.vue` 的移动端全屏表单版本：

- 全屏表单替代弹窗对话框
- 简化为核心字段：名称、主机、端口、用户名、认证方式、密码/密钥
- 高级选项（跳板机链路、端口转发、tmux、git 同步）折叠在"高级设置"展开区
- 分组选择用简单 Picker 替代树形选择
- 表单验证和错误提示适配触摸交互

### 1.2 服务器操作

- 长按服务器 → 操作菜单（编辑/删除/复制）
- 向左滑动 → 快速删除（确认对话框）
- 编辑复用 MobileConnectForm

## 二、移动端 SFTP

### 2.1 MobileSftpView.vue

**新建文件**: `src/components/mobile/MobileSftpView.vue`

全屏 SFTP 文件浏览器（非桌面端的分屏面板）：

- 复用现有 composables: `useTabSftp`、`useSftpPane`
- 单列文件列表，触摸优化行高
- 导航栏显示当前路径，支持面包屑导航
- 从终端页通过按钮进入 SFTP 页

### 2.2 文件操作

| 操作 | 交互方式 |
|------|---------|
| 打开目录 | 点击 |
| 下载文件 | 长按 → 操作菜单 |
| 上传文件 | 工具栏按钮 → 原生文件选择器 |
| 重命名 | 长按 → 操作菜单 |
| 删除 | 长按 → 操作菜单（确认对话框） |
| 修改权限 | 长按 → 操作菜单 |

操作菜单使用 Bottom Sheet / Action Sheet 替代桌面端右键菜单。

### 2.3 文件选择器

上传文件时使用原生文件选择器：
- 桌面端: `rfd` (已有)
- 移动端: `tauri-plugin-dialog`（支持 iOS/Android 原生文件选取）

## 三、移动端设置

### 3.1 MobileSettings.vue

**新建文件**: `src/components/mobile/MobileSettings.vue`

精简设置页面，分组展示：

| 分组 | 设置项 |
|------|--------|
| 终端外观 | 字号、字体、主题 |
| 语言 | 中文/English |
| AI 供应商 | Provider 选择、API Key 配置 |
| 安全 | 主密码设置 |
| 关于 | 版本信息、开源协议 |

移动端跳过的设置：
- 快捷键自定义（无物理键盘）
- SFTP 布局模式（移动端固定全屏）
- 侧边栏动画
- 本地 AI 模型管理（移动端不支持）

## 四、系统 API 替换

### 4.1 剪贴板

| 现有 | 替换为 | 说明 |
|------|--------|------|
| `arboard` crate | `tauri-plugin-clipboard-manager` | 支持 iOS/Android |

**修改文件**: `src-tauri/src/commands/clipboard.rs`

- 桌面端保留 `arboard`（feature gate）
- 移动端通过 Tauri 插件调用系统剪贴板

### 4.2 文件对话框

| 现有 | 替换为 | 说明 |
|------|--------|------|
| `rfd` crate | `tauri-plugin-dialog` | 支持移动端原生文件选取 |

**修改文件**: `src-tauri/src/commands/local_fs.rs`

### 4.3 URL 打开

| 现有 | 替换为 | 说明 |
|------|--------|------|
| `open` crate | `tauri-plugin-opener` | 支持移动端 URL 打开 |

**修改文件**: `src-tauri/src/commands/local_fs.rs`、`src-tauri/src/commands/update.rs`
